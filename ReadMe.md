## Description
CampusMart is a simple e-commerce web application built using HTML, CSS, and JavaScript.  
It allows users to browse products, register, log in, and contact customer care.  
This project is developed as part of the Web Technology lab assignment.

---

## Features
- Navigation bar with links to Home, Login, About, and Contact pages  
- User Registration and Login with form validation  
- Product listing with Wishlist and Order options  
- Contact form for customer support  
- Clean, responsive user interface  

---

## Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Storage: LocalStorage (for demo purposes)  
- Platform: GitHub Pages (for hosting)

---

## How to Run
1. Download or clone this repository.  
2. Open the folder in VS Code.  
3. Right-click on `index.html` → choose **Open with Live Server**.  
4. Navigate between pages using the top menu.
